Configure SIP terminals

The only thing we require is to have 2 SIP terminals (hardphone, softphone, Android/IOS APP) and configure them as follows:

ALICE

  • User: alice

  • Password: alice

  • Domain”: users.democompany.com (or the IP address if we are using the domain trick)

BOB

  • User: bob

  • Password: bob

  • Domain”: users.democompany.com (or the IP address if we are using the domain trick)

Tip

It’s possible that both the user and the domain are requested together, so we have to use alice@users.democompany.com and bob@users.democompany.com (or the IP address if we are using the domain trick).

After configurint the terminals, Alice should be able to call Bob by only dialing 102 in its terminal.
